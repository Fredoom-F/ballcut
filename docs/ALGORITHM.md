# 当前 OpenCV 识别判据

更新日期：2026-06-19  
代码基线：`analyzer/analyze_video.py`

本文记录当前可复现阈值，不代表人体姿态、球拍检测或裁判级判断。

## 分析采样

- 原视频按比例缩小到最大宽度 480px。
- 快速、标准、精细模式分别目标采样 8、12、15fps。
- 使用 `grab/retrieve` 跳过非采样帧，避免完整解码所有帧。
- 速度与加速度单位基于缩小后的分析画面，不是原始 4K 像素。

## 球候选

1. 按运动类型建立 HSV 球色范围。
2. 当前帧与上一帧灰度差大于 18 时记为运动区域。
3. 球色区域与运动区域取交集。
4. 轮廓面积至少 2px，且不能超过运动对应的最大画面面积比例。
5. 候选宽或高不能超过画面的 22%。
6. 按圆度 35%、面积 27%、预测位置距离 28%、边缘距离 10% 加权。
7. 首个候选得分至少 0.32；已有轨迹时，单次跳跃不能超过画面对角线的 22%。
8. 连续丢失约 0.5 秒后终止当前轨迹，避免跨场景误连。
9. 对外展示的轨迹点还必须具备局部实际位移，最低速度为分析画面对角线的 2.5%，用于过滤原地闪烁灯光。

用户校准球颜色时，点击区域采样中位 RGB，再转换为 HSV：

- 色相上下扩展 11。
- 饱和度下限至少 45，亮度下限至少 55。
- 缓存键包含校准颜色。

## 镜头稳定

相邻帧使用相位相关估算全局位移，并计算运动像素面积。事件附近窗口内：

| 机位 | 窗口 | 位移 75 分位上限 | 运动能量 65 分位上限 |
| --- | ---: | ---: | ---: |
| 自动 | ±0.55 秒 | 2.0px | 0.25 |
| 底线固定 | ±0.60 秒 | 1.8px | 0.23 |
| 边线广角 | ±0.50 秒 | 2.0px | 0.25 |
| 手持跟拍 | ±0.38 秒 | 3.0px | 0.30 |

手持模式容忍更大镜头位移，因此可能提高召回，也会增加候选量。

## 疑似击球

候选必须同时满足：

- 前后 5 个轨迹点相邻间隔均不超过 0.55 秒。
- 球的前后最大速度超过最低门槛。
- 镜头稳定。
- 球靠近人体尺度活动区域代理。
- 球位于运动允许的画面区域。
- 网球、羽毛球、乒乓球必须出现方向反转。
- 篮球、足球、高尔夫允许方向反转或显著加速。

灵敏度阈值按分析画面对角线 `D` 缩放：

| 灵敏度 | 最低速度 | 加速度阈值 | 方向余弦 | 约等于转向角 |
| --- | ---: | ---: | ---: | ---: |
| 保守 | `0.11D` | `0.22D` | `< -0.62` | `> 128°` |
| 标准 | `0.09D` | `0.18D` | `< -0.48` | `> 119°` |
| 灵敏 | `0.07D` | `0.14D` | `< -0.32` | `> 109°` |

显著加速还要求前后速度倍率大于 2.2。相邻击球不足 0.65 秒时只保留置信度更高者。

击球置信度：

```text
0.30
+ 方向反转强度 × 0.32
+ 加速度强度 × 0.25
+ 轨迹连续度 × 0.13
```

最终限制在 0～0.96。若击球后速度超过击球前 2 倍、球位于画面较上方
`y <= 0.62`，且运动为网球或羽毛球，则建议标为发球。

## 等待与捡球片段

有效区间来自三类信号：

- 稳定镜头下球速不低于 `0.05D`。
- 击球事件前后保护区。
- 高于视频自适应运动能量阈值的稳定画面活动。

自动删除强度：

| 强度 | 击球保护区 | 最短等待片段 |
| --- | ---: | ---: |
| 保守 | ±3.2 秒 | 10.0 秒 |
| 标准 | ±2.2 秒 | 6.5 秒 |
| 激进 | ±1.4 秒 | 4.0 秒 |

若删除区间内至少 28% 样本满足全局位移大于 2px 或运动能量大于 0.32，
原因标为“镜头移动或拍摄中断”；否则标为“低运动量且未追踪到球”。

两个删除区间中间若只有不超过 2 秒的保留段，且没有击球事件，会合并为一个删除段。

## 精彩片段

- 后端只为置信度至少 0.58 的击球生成精彩候选。
- 初始范围为击球前 2.2 秒至后 3 秒。
- 前端精彩分继续融合置信度、击球后画面球速、方向变化、轨迹连续度、附近连续击球、人工确认和收藏。
- 最终是否进入导出还受用户精彩阈值、集锦时长和逐段勾选影响。

## 轨迹特效

- 预览和导出只绘制最近一次击球时刻之后 1.2 秒内的轨迹。
- 发球抛球发生在击球前，因此不会进入轨迹特效。
- 相邻轨迹点时间间隔超过 0.35 秒，或归一化画面位置跳跃超过 0.22 时立即截断，避免连到背景灯光。

## 拍摄质量提示

- 轨迹覆盖低于 10%：建议校准球颜色。
- 清晰度中位数低于 45：提示画面偏模糊。
- 亮度中位数低于 55：提示画面偏暗。
- 镜头稳定比例低于 65%：建议固定机位或三脚架。

## 解释边界

- 活动区域是帧差轮廓形成的人体尺度代理，不是人体检测框。
- 当前没有人体姿态、球拍或球网语义模型。
- “候选命中率”来自用户确认/忽略，能衡量已复核候选的误报情况，不能直接当作召回率。
- 不同机位、球色、分辨率和光照需要分别复核，不能把单段视频结果外推为通用准确率。

# Project Skill

Use this skill when a task needs reusable project knowledge, repeatable workflow steps, or local helper scripts.

## When To Use

- A task repeats across the project.
- A task needs project-specific context.
- A workflow benefits from a saved checklist, template, or script.

## Workflow

1. Read `skills/context.md` for project background.
2. Check `skills/scripts/` for reusable helpers.
3. Apply the smallest workflow that solves the task.
4. Update this skill only when a new repeatable practice is discovered.

## Output Standard

- Keep results practical and directly usable.
- Capture reusable lessons in `skills/context.md`.
- Keep one-off notes out of the skill unless they become repeatable.

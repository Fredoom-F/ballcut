# 剪球

剪球是一个面向球类运动爱好者的本地 AI 运动剪辑 MVP。当前版本可以在浏览器里完成视频载入、运动类型选择、冗余片段识别演示、击球/关键动作特效预览、精彩瞬间时间线、带水印预览导出和剪辑决策 JSON 下载。

当前实现是本地最小价值单元：视频不会上传到外部服务器，页面和处理逻辑都运行在 `127.0.0.1`。

## 运行方式

进入应用目录：

```powershell
cd app
npm start
```

然后打开：

```text
http://127.0.0.1:4173/
```

## 当前功能

- 本地视频上传与预览
- 生成演示视频，方便无素材时测试流程
- 运动类型选择：网球、羽毛球、乒乓球、篮球、足球、高尔夫
- 剪辑目标选择：综合剪辑、删除捡球等待、精彩瞬间、击球特效
- 自动生成击球/关键动作点、冗余片段、精彩片段
- 播放时跳过冗余片段
- Canvas 击球特效预览
- 导出带水印 WebM 预览视频
- 下载剪辑决策 JSON

## 项目结构

```text
app/
  index.html   页面结构
  styles.css   界面样式
  app.js       本地视频分析演示、时间线、特效和导出逻辑
  server.js    本地静态服务
```
